# 网络流量分析与异常检测系统

一个基于Python实现的网络流量分析与异常检测系统，支持从CSV文件读取网络流量数据，构建图结构进行分析，并提供可视化Web界面。

## 功能特性

### 基本功能（2.3）
1. 数据读取：从CSV文件读取网络流量数据
2. 图结构构建：使用邻接表构建网络拓扑图
3. 流量排序：按流量大小排序节点
4. 筛选与排序：
   - HTTPS连接筛选
   - 单向流量占比>80%的恶意节点筛选
5. 路径查找：拥塞程度最小和跳数最小路径

### 扩展功能（2.4）
1. 星型结构检测
2. 安全规则检测

### 升级功能（2.5）
1. Web界面
2. 图可视化

## 安装

```bash
pip install -r requirements.txt
```

## 使用

运行Web服务器：

```bash
python app.py
```

然后在浏览器中访问 http://localhost:5000
