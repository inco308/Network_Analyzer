
# 网络流量分析器 - 安装和使用说明

## 系统要求

- 操作系统：Windows 10/11 (64位)
- 无需安装Python或其他依赖

## 安装说明

1. 下载并解压本程序包
2. 将解压后的文件夹放置在任意位置
3. 双击运行 `网络流量分析器.exe` 即可启动程序

## 文件说明

程序包包含以下文件：

- `网络流量分析器.exe` - 主程序（图形界面）
- `network_analyzer.exe` - C语言后端分析程序
- `network_data.csv` - 示例网络流量数据文件
- `README_使用说明.md` - 本文档

## 使用说明

### 1. 启动程序

双击运行 `网络流量分析器.exe`

### 2. 选择数据文件

- 点击"选择文件"按钮，选择您的网络流量CSV文件
- 或者使用示例数据 `network_data.csv`

### 3. 主要功能

#### 基本功能

1. **获取图信息** - 显示网络拓扑图的基本信息（节点数、边数、会话数）
2. **流量排序** - 按流量大小对节点进行排序
3. **HTTPS筛选** - 筛选出所有HTTPS连接的节点
4. **可疑节点** - 查找单向流量占比大于指定阈值的节点

#### 高级功能

5. **路径查找** - 查找两个节点之间的最短路径
6. **星型结构** - 检测网络中的星型拓扑结构
7. **获取子图** - 获取指定节点所在的子图并可视化
8. **子图列表** - 列出所有连通子图
9. **安全规则检查** - 检查网络流量是否符合安全规则

#### 数据处理功能

- **PCAP转CSV** - 将PCAP格式的抓包文件转换为CSV格式
- **实时捕获** - 实时捕获网络流量并保存为CSV文件

### 4. 参数说明

- **最小占比 (可疑节点)**: 用于判断可疑节点的单向流量阈值，默认0.8（80%）
- **最小边数 (星型结构)**: 星型结构检测的最小边数，默认5
- **源IP / 目的IP**: 用于路径查找的源和目的IP地址
- **安全规则检查参数**: 用于安全规则检查的目标地址和地址范围

### 5. 图可视化说明

点击"获取子图"后，程序会：
1. 自动生成子图的HTML可视化文件
2. 使用默认浏览器打开可视化页面
3. 支持鼠标滚轮缩放、拖拽平移
4. 悬停可查看完整IP地址

### 6. 数据格式要求

CSV文件应包含以下列：
- source_ip: 源IP地址
- dest_ip: 目的IP地址
- protocol: 协议号（TCP=6, UDP=17等）
- src_port: 源端口
- dst_port: 目的端口
- data_size: 数据大小（字节）
- duration: 持续时间（秒）

## 常见问题

**Q: 程序无法启动？**
A: 请确保您使用的是64位Windows系统。

**Q: 实时捕获功能无法使用？**
A: 请确保已安装Npcap或Wireshark，并以管理员身份运行程序。

**Q: 可视化页面无法打开？**
A: 请确保电脑上安装了现代浏览器（Chrome、Edge、Firefox等）。
